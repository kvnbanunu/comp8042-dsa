# How to run code

For each section you can run make &lt;section&gt; or java &lt;ProgramName&gt;.java

## Binary Search Tree Check

```sh
make bst
```

or

```
java BSTChecker.java
```

## AVL Tree Contact List

```sh
make avl
```

or

```
java ContactList.java
```

## Social Network

```sh
make graph
```

or

```
java FriendGraph.java
```
